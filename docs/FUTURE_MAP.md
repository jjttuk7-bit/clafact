# 수직 진화 사다리 — L1 → L4

작성: 2026-07-20 · 원칙: **각 단계는 전 단계의 증명 없이 진입할 수 없다.** 사다리는 건너뛰라고 있는 게 아니다.

## 한눈에

```
L1 PoC ──"5축 실측 정밀도"──▶ L2 도구 ──"주간 능동 사용자"──▶ L3 서비스 ──"유료 갱신율"──▶ L4 인프라
```

## L1 — PoC (현재 위치)

특정 언론(조선일보)·특정 통계원(KOSIS)에서 파이프라인이 작동함을 증명하는 단계.

| 항목 | 내용 |
|---|---|
| 핵심 지표 | 골든셋 250건에서 판정 정확도 + 판단불가 적중률 (run_eval 최종 수치) |
| 진입 조건 | — (진행 중) |
| 최소 팀 | 현 구성 그대로: 인간 4 + Hermes |
| 기술 장벽 | 28만 표 실검색 매핑 정밀도 (경로 C가 픽스처 성적을 실세계에서 재현하는가) |
| 비기술 장벽 | 라벨링 인력 시간 (골든셋 250건 = 사람만 가능) |

**L2 진입 증명**: 5축 전부 실 KOSIS API 검증 완료 + 대량 배치(2,649건) 처리 리포트 + 릴리스 게이트 전항목 PASS.

## L2 — 도구 (셀프서브)

기자·리서처가 기사 URL을 넣으면 수치 검증 리포트가 나오는 도구. **씨앗은 이미 있다**: Hermes 모닝 브리프 크론잡(자동 실행 패턴), 서비스층 CLI(ingest→process→report가 곧 리포트 생성기), Streamlit 데모(입력 UI).

| 항목 | 내용 |
|---|---|
| 핵심 지표 | **주간 능동 사용자** (팀 밖의 사람이 스스로 다시 쓰는가) |
| 최소 팀 | 2명 (유지보수 1 + 사용자 피드백 처리 1) — 사다리 중 가장 작은 팀으로 가능 |
| 기술 장벽 | URL → 기사 본문 추출의 매체 다양성 (크롬 제거가 매체별 수작업이면 확장 불가) |
| 비기술 장벽 | 기자들의 작업 흐름에 들어가기 — 도구가 좋아도 마감 루틴 밖이면 안 쓴다 |

**L3 진입 증명**: 외부 사용자 N명이 4주 연속 사용 + 검증 리포트에 대한 정정 요청률이 임계 이하 (오판정이 신뢰를 깎지 않음을 증명).

## L3 — 서비스 (첫 수익 실험)

언론사·미디어 스타트업 대상 API, 또는 뉴스레터 "오늘의 수치 검증". 여기서 첫 과금 실험 — 다른 프로젝트의 수익화 스터디(구독·API 과금 모델)와 접점.

| 항목 | 내용 |
|---|---|
| 핵심 지표 | **유료 전환·갱신율** (한 번 사는 게 아니라 계속 사는가) |
| 최소 팀 | 3~4명 (엔진 + 영업/CS + 운영). HITL 리뷰어가 처음으로 "비용"이 됨 |
| 기술 장벽 | SLA — 실시간성·가용성. 배치 도구와 유료 API는 다른 생물 |
| 비기술 장벽 | 언론사 조달 관행 + "검증 결과에 대한 법적 책임" 계약 조항 (판단불가 클래스와 감사 로그가 방어선) |

**L4 진입 증명**: 유료 고객이 우리 골든셋·매핑 테이블을 "자기 데이터로도" 쓰고 싶다고 요구하기 시작함 — 엔진이 아니라 자산에 수요가 붙는 신호.

## L4 — 인프라 (공용 레이어)

한국어 통계 Claim 검증의 공용 레이어. 코드가 아니라 **골든셋·별칭 사전·매핑 테이블·규칙 카드 자체가 판매/공유 자산**이 되는 단계. 공공데이터 활용 프레임워크(PDL)와의 접점: 검증 이력·매핑 테이블의 표준화 공개가 공공재 포지션을 만든다.

| 항목 | 내용 |
|---|---|
| 핵심 지표 | 제3자 시스템의 우리 자산 호출량 (API/데이터셋 라이선스 건수) |
| 최소 팀 | 6명+ (표준·거버넌스·파트너십이 엔지니어링만큼 커짐) |
| 기술 장벽 | 매핑 테이블의 버전 관리·호환성 (KOSIS 개편 5년 주기·기관 개명이 자산을 썩게 함 — A2-0011 계열 자동화 필수) |
| 비기술 장벽 | 중립성 거버넌스 — 인프라가 되는 순간 "누가 골든셋을 정하는가"가 정치적 질문이 됨 (SNU팩트체크의 종말이 바로 이 지점이었음을 기억할 것) |

## 사다리 전체를 관통하는 한 가지

각 단계에서 파는 것이 바뀐다: **L1 증명 → L2 시간 절약 → L3 신뢰의 SLA → L4 자산 접근권.**
그러나 밑천은 하나다 — 플라이휠이 쌓는 골든셋·규칙·별칭. 그래서 어떤 단계에서든
"골든셋에 사람 라벨만"이라는 가드레일이 무너지면 사다리 전체가 무너진다.
