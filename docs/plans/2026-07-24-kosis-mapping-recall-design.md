## 문제

실 KOSIS 검색은 분류 지표어 하나만 사용해 동일 지표의 연간·월간·연령별 표를 구분하지 못한다. 또한 화면은 근거 행이 없으면, 감사 로그에 선택 통계표가 있어도 “표를 찾지 못했다”고 표시한다.

## 결정

분류 도메인별 짧은 보조 검색어를 최대 하나 더 사용한다. 예: 출생아는 `출생아`와 `인구동향`, 실업률은 `실업률`과 `경제활동인구`를 검색한다. 결과를 표 ID로 중복 제거하고, 주장에 등장한 핵심어와 월간·연간 적합도를 포함해 재정렬한다. 따라서 Claim당 통합검색 호출은 최대 두 번이며, 수치 조회 호출은 기존과 같다.

`1~8월` 같은 누적 월 범위는 기사 작성 연도의 마지막 월로 정규화한다. 이는 월간 표 우선순위에 사용하며, 누적값 계산 자체를 보장하지 않는다.

화면은 수치 근거가 없더라도 감사 로그의 선택 표·검색어를 표시한다. “통계표를 찾지 못함”은 감사 로그에도 표 정보가 없을 때만 표시한다.
