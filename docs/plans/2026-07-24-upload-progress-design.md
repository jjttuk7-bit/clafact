# 기사 등록 진행 상태 표시 설계

## 목표

CSV 업로드 후 기사 등록 과정의 진행 상황을 전역 Streamlit 실행 표시가 아닌 운영 실행 영역 안에서 보여준다.

## 설계

- 기사 등록 버튼을 누르면 운영 실행 영역에 진행 상태 카드가 나타난다.
- 단계는 `파일 읽기 → 기사 등록 → 출처 분류 → 검증 후보 준비` 순서로 표시한다.
- 현재 단계는 진행 중, 완료 단계는 완료 상태로 표시한다.
- 각 단계가 끝날 때 기사 수, 문장 수, Claim 수, KOSIS 후보 수를 표시한다.
- 등록 성공 시 기존 업로드 요약과 결과 화면을 유지한다.
- 예외 발생 시 진행 카드 안에 실패 단계와 오류 내용을 표시한다.
- 전체 새 정책 재분류 버튼은 노출하지 않는다.

## 구현 범위

- `streamlit_app.py`: 기사 등록 핸들러에 단계 상태 UI와 오류 상태를 추가한다.
- `tests/test_upload_scoped_dashboard.py`: 운영 실행 영역에 단계 라벨과 상태 표시 호출이 존재하는지 검증한다.
