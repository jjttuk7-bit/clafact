## Goal

상단 탭을 고정 왼쪽 내비게이션으로 전환해 운영 화면의 정보 위계와 본문 집중도를 높인다.

## Layout

- 데스크톱: 약 230px의 사이드바에 ClaFact 브랜딩과 운영 홈·검증·검증자 리뷰·플라이휠·자산 현황 메뉴를 세로로 배치한다.
- 활성 메뉴: 청록 포인트 바, 대비 표면, 명확한 글자색으로 표시한다.
- 본문: 현재 선택된 화면만 렌더링하고 기존 기능·위젯 키·데이터 흐름은 유지한다.
- 좁은 화면: 사이드바를 감추고 상단 선택 메뉴로 동일한 화면을 전환한다.

## Accessibility

- 메뉴는 라디오 선택 컨트롤을 사용해 키보드 접근과 현재 선택 상태를 보장한다.
- 활성 상태는 색상 외에도 텍스트와 표면 차이로 전달한다.
- 라이트·다크 표면 토큰을 공유한다.
