## Mission

Evidence에 기반해 Verdict를 판단하고 서비스 발행 기준을 관리한다.

## Owns

- Verdict 정책
- 판단불가 사유
- 발행등급 기준

## Rules

- Evidence 없이 Verdict를 내리지 않는다.
- 근거가 부족하면 판단불가로 남기고 부족한 근거를 설명한다.
- Daily 기록과 확정 정책을 구분한다.
