## Owns

- KOSIS 테이블 매핑
- Evidence 검색 전략
- 검색·리랭커 결과와 근거 기록

## Rules

- 출처, 조회 시점, 테이블·지표 식별자를 기록한다.
- 검색 결과와 확정 근거를 구분한다.
- 민감정보와 내부 접근 링크를 기록하지 않는다.
