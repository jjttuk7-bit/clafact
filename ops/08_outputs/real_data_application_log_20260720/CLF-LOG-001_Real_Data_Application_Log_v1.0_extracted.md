6. 이 기록이 증명하는 것 
① 테스트·Mock은 목적지가 아니지만, 실전으로 갈 때 돈과 시간을 아끼는 안전벨트다. 
dry-run이 예산을 한 푼도 쓰지 않고 쓰레기 질의를 먼저 보여줬고, 30건 소량 배치가 795회를 지르기 전에 전략 오
류를 잡았다. 검증 없이 전량 실행했다면 예산의 80%를 날렸을 것이다. 
② HITL 발행규칙은 장식이 아니었다. 
첫 실판정 5건의 오'불일치'는 "불일치는 무조건 사람 승인" 규칙 덕분에 자동 발행되지 않았을 것이다. 설계가 최악
의 사고를 막았다. 
③ "실패 1건 = 자산 1줄"이 실제로 돌았다. 
오판 5건이 규칙 카드 3종과 리랭커로 전환됐고, 각 카드에는 유래 사례가 그대로 기록돼 있다. 같은 실패를 두 번 하
지 않는 구조가 실데이터에서 작동했다. 
7. 남은 과제 
• 모집단 정렬: "175개 품목 평균"(부분집합)을 전체 CPI와 대조하는 문제 — 규칙 A2-0017 후보. 7차에서 유일하게 
남은 오'불일치' 
• 지수 기준연도 표기 오파싱: "소비자물가지수는 115.71(2020년=100)"에서 기준연도 표기를 주장 시점으로 읽어 
2020년 자료를 찾는다 — 판단불가로 안전하게 빠지지만 사유가 틀렸다 
• 매핑 적합성 확대: 리랭커는 물가 계열에서 검증됐고, 나머지 지표(취업자·출생아·인구)로 확대 필요 
• 판정률 회복: 가드를 쌓아 오판은 막았으나 판단불가 비중이 높다. 정직한 회피는 좋은 기본값이지 최종 목표가 
아니다
